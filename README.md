# 📂 File Organizer & Duplicate Cleaner

A Python-based application that automatically organizes files into folders based on their file type and detects duplicate files using MD5 hashing. The project includes a simple and interactive Streamlit interface.

## 🚀 Features

- 📁 Automatically organizes files into categories
- 🖼️ Supports Images, Documents, Videos, and Archives
- 🔍 Detects duplicate files using MD5 hashing
- 🗂️ Moves duplicate files into a separate folder
- 💻 Simple Streamlit user interface

## 🛠️ Technologies Used.

- Python
- Streamlit
- OS
- Shutil
- Hashlib

## 📁 Project Structure

```
File-Organizer-Duplicate-Cleaner/
│── app.py → UI (Streamlit)
│── engine.py → logic (file organize + duplicate detect)
```

## ▶️ How to Run

1. Install Streamlit

```bash
pip install streamlit
```

2. Run the application

```bash
streamlit run app.py
```
